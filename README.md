
# CiefpTMDBSearch v1.0

**The most beautiful, fastest and smartest TMDB plugin for Enigma2 – ever made.**

![CiefpTMDBSearch](https://raw.githubusercontent.com/ciefp/CiefpTMDBSearch/main/screenshots/main.jpg)

Search movies and TV shows directly from your Enigma2 receiver using **The Movie Database (TMDB)** – with high-quality posters, ratings, cast, genres and full plot summary in **14 languages**!

### Features
- Gorgeous full-screen design with background & big poster
- Multi-language support (14 languages including Serbian, Croatian, Bosnian, Slovak, Russian, Arabic, Chinese…)
- Poster caching (`/tmp/CiefpTMDBSearch/`) for instant loading
- Lightning-fast search and info loading
- Shows up to 5 main actors with character names
- Rating, year, runtime, genres, director/creator
- Blue button in Settings cycles through languages
- Works perfectly on OpenATV, OpenBH, OpenPLI, VTI, Egami, OpenVision…

### Supported Languages

- 中文 / Chinese

### Installation
1. Copy the `.ipk` file to your receiver (FTP/DCC)
2. Install via Menu → Setup → Software Management → Install local extension (or IPK Installer plugin)
3. Launch from Plugins menu
4. First run: press **YELLOW** button → enter your free TMDB API key  
   Get your free key here: https://www.themoviedb.org/settings/api
